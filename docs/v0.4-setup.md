# javahome 0.4 setup wizard

Version 0.4 adds a guided setup workflow and safer profile management.

## Setup wizard

Run:

```bash
javahome setup
```

The wizard guides the user through:

1. shell detection
2. Java installation selection
3. activation mode selection
4. optional `jhome` helper installation
5. optional shell completion installation
6. optional `javahome doctor` run

Useful variants:

```bash
javahome setup --shell bash
javahome setup --shell zsh
javahome setup --shell fish
javahome setup --shell powershell
javahome setup --dry-run
```

## Automatic backups

Profile edits now create timestamped backup files when the target profile already exists.

Example backup names:

```text
~/.bashrc.javahome-backup-20260708-084500
~/.zshrc.javahome-backup-20260708-084500
```

Backups are created before profile updates and before removing javahome blocks.

## Uninstall

Remove javahome-managed blocks from the detected shell profile:

```bash
javahome uninstall
```

Remove from a specific profile:

```bash
javahome uninstall --shell bash
javahome uninstall --shell zsh
javahome uninstall --shell fish
javahome uninstall --shell powershell
```

Preview first:

```bash
javahome uninstall --shell bash --dry-run
```

Clean all supported shell profiles:

```bash
javahome uninstall --all
```

## Notes

`javahome uninstall` only removes marked javahome blocks. It does not delete JDK installations and does not remove manually written shell configuration.
